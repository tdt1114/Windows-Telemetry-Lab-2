# Windows Endpoint Telemetry & SOC Triage (Splunk Cloud)

## Overview
This project demonstrates ingestion of Windows endpoint telemetry into Splunk Cloud and SOC-style alert triage, investigation, and documentation.

## Skills Demonstrated
- SIEM ingestion (Splunk Cloud, HEC)
- Endpoint telemetry analysis (Windows Security, Sysmon)
- Alert triage and false positive determination
- Detection tuning and documentation

## Project Artifacts
- SOC alert investigation case study
- Lab design and ingestion notes
- SPL detection and analysis queries

See `/lab-notes` for detailed investigation and analysis.
